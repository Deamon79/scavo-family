# The Scavo Family — landing page

A private one-page introduction shared with potential landlords.

Made with care; static HTML / CSS / JS, no build step.

- `index.html` — page structure and copy
- `style.css` — design system, layout, motion
- `main.js` — small enhancements (year, scroll reveal, marquee pause)
- `images/` — family photos, EXIF-stripped and resized
- `robots.txt` — blocks indexing

To preview locally:

```bash
python3 -m http.server 4747
# open http://localhost:4747
```

Hosted on GitHub Pages.
